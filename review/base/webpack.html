什么是 webpack？

webpack是

模块加载器

兼

模块打包工具

为什么用webpack?

他的目的就是把有依赖关系的各种文件打包成一系列的静态资源,

把各种资源，如

JS（JSX）,

coffee,

css(less,sass),

图片

等都作为

模块

来使用和处理

怎么用webpack？

配置环境

检查基础环境：1

node -v

npm -v

：2检查webpack打包环境

npm i === npm install

npm install webpack -g

npm install webpack -cli -g

webpack -v （检查安装是否成功）

npm init -y (默认创建package.json文件)


打包方式

1.

npx webpack ./src/xxx.(后缀名)(目标文件的地址)

2.

npx webpack --config webpack.config.js

3.

npm run build

第3种方法就是将第二种方法的命令放到通过

npm init

生成的package.json文件里的

script

标签中
