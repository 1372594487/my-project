mysql事务处理：一组mysql语句的集合，可以同时执行多条sql语句.所有sql语句都执行成功，事务处理成功，如果有一条失败，事务处理失败，事务回滚至初始状态
事务处理的处理结果(原子性)：要么全部成功，要么全部失败。

原子性，一个原子事务要么完整执行，要么不执行
一致性，一致性代表了底层数据存储的完整性。
隔离性，隔离性意味着事务必须在不干扰其他进程或事物的前提下独立执行。
持久性，持久性表示在某个事物的执行过程中，对数据所做的所有改动都必须在事务成功结束前保存至某种物理存储设备
